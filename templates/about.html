AI Image Detector에 대한 자세한 정보를 확인하세요

📋 프로젝트 개요

AI Image Detector는 최신 딥러닝 기술을 활용하여 AI 생성 이미지와 실제 이미지를 구분하는 고정밀 탐지 시스템입니다. Vision Transformer(ViT) 모델을 기반으로 하여 99% 이상의 정확도를 달성하며, 사용자 피드백을 통한 지속적인 학습과 개선이 가능한 적응형 AI 시스템입니다.

🎯 목표

AI 생성 이미지와 실제 이미지를 정확하게 구분하여 디지털 미디어의 신뢰성을 보장합니다.

🔬 기술

Vision Transformer, PyTorch, Flask를 활용한 최신 AI 기술 스택을 사용합니다.

📊 성능

99.42% 전체 정확도, 100% REAL 탐지율, 99.07% FAKE 탐지율을 달성했습니다.

🔄 학습

사용자 피드백을 통한 지속적인 모델 개선과 성능 향상을 지원합니다.

🚀 개발 과정

1단계
기본 모델 구현
Hugging Face의 사전 훈련된 ViT 모델을 활용하여 기본적인 AI 이미지 탐지 시스템을 구축했습니다. 초기 정확도는 72.67%였습니다.
2단계
데이터 분석 및 문제점 파악
test2 데이터셋을 활용한 종합적인 성능 테스트를 통해 REAL 이미지 탐지 성능이 50%로 낮다는 문제점을 발견했습니다.
3단계
모델 재훈련
test2 데이터셋을 사용하여 모델을 재훈련하고, 데이터 증강을 통해 성능을 크게 향상시켰습니다. 전체 정확도가 99.42%로 향상되었습니다.
4단계
웹 애플리케이션 개발
Flask를 활용한 반응형 웹 애플리케이션을 개발하고, 사용자 피드백 시스템을 구축했습니다.
5단계
고도화 및 최적화
상세한 분석 설명, 통계 시스템, 반응형 디자인 등 사용자 경험을 향상시키는 기능들을 추가했습니다.

🛠️ 기술적 특징

🧠 AI 모델

  • Vision Transformer (ViT) 아키텍처
  • 224x224 고해상도 이미지 처리
  • 중앙 크롭 전처리 기법
  • Softmax 확률 분포 출력

📊 데이터 처리

  • 다양한 이미지 포맷 지원
  • 실시간 전처리 최적화
  • 메모리 효율적인 배치 처리

🌐 웹 기술

  • Flask 웹 프레임워크
  • 반응형 CSS Grid/Flexbox
  • JavaScript ES6+ 기능
  • SQLite 데이터베이스

🔄 학습 시스템

📖 사용 방법

  1. 이미지 업로드: 홈페이지에서 분석하고 싶은 이미지를 업로드합니다.
  2. 자동 분석: AI 모델이 이미지를 분석하여 FAKE 또는 REAL로 분류합니다.
  3. 결과 확인: 분류 결과와 상세한 분석 설명을 확인합니다.
  4. 피드백 제공: 결과가 정확한지 피드백을 제공하여 모델 개선에 기여합니다.
  5. 통계 확인: 통계 페이지에서 전체적인 성능과 피드백 현황을 확인합니다.
